## 🎯 主要機能

- ひらがなを「偏＋旁」の暗号文に変換（例：「てき」→「身白 土黒」）
- 暗号文から元のひらがなを復号（例：「身白 土黒」→「てき」）
- 漢字風合成画像で視覚的にも出力
